# Actix-Angular-Web

A blog for myself build with actix-web and angular.

## Usage

### Production

```shell
cd ./app
ng build
cd ..
cargo run --release
```

### Develop

```shell
#ng
cd ./app
ng serve
#actix
cargo run
```
